# Brainfuck.helm
A brainfuck interpreter implemented by Helm Template Language.

Only run correctly with first example from [eso-brainfuck](https://esolangs.org/wiki/Brainfuck)

# requirement
Helm >= 3.1.2

# Usage
```
# clone this repo and then 
helm template .
```
